# cwmock
##### command-line werkzeug http/rest/json mock

`python setup.py install`

`cwmock --help`

`cwmock -f file.json`

`cwmock -f file.xml -c 201`

`cwmock -f path/file.txt -c 201 -e myendpoint -p 8080`
