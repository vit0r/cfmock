##### command-line werkzeug http/json/xml/plain-text mock

`python setup.py install`

`cwmock --help`

`cwmock -f file.json`

`cwmock -f file.xml -c 201`

`cwmock -f path/file.txt -c 201 -e -e user/<int:id> -p 8080`
